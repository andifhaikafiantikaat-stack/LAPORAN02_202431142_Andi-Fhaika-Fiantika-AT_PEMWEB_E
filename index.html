a. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?

Dalam pemrograman web modern, HTML (HyperText Markup Language) berfungsi sebagai tulang punggung untuk struktur dan konten. Namun, dua teknologi lainnya sangat penting untuk menciptakan pengalaman pengguna yang memuaskan:

  • CSS (Cascading Style Sheets): Digunakan untuk mengontrol penyajian (presentasi) visual halaman web, termasuk tata letak, warna, tipografi, dan responsivitas. Tanpa CSS, halaman web akan terlihat polos dan tidak menarik. CSS memisahkan desain dari struktur, membuat kode lebih bersih dan mudah dikelola.
  • JavaScript (JS): Digunakan untuk menambahkan perilaku (behavior) dan interaktivitas pada halaman web. JS memungkinkan adanya fitur dinamis seperti validasi formulir, manipulasi DOM (Document Object Model) untuk mengubah konten secara real-time, animasi kompleks, dan komunikasi asinkron dengan server (misalnya, AJAX/Fetch API).

Kombinasi ketiganya (HTML, CSS, JS) dikenal sebagai Tumpukan Standar Web, yang memungkinkan pembuatan aplikasi web yang kaya fitur, indah, dan responsif.

b. Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

Responsive Web Design (RWD) adalah pendekatan dalam desain dan pengembangan web yang bertujuan untuk membuat situs web merespons dan beradaptasi dengan lingkungan penggunanya—terutama ukuran layar, orientasi, dan platform—menggunakan tata letak yang fleksibel (fluid grid), gambar yang fleksibel (fluid image), dan kueri media (media queries) CSS.

Tujuan utama RWD adalah:

  • Konsistensi Pengalaman Pengguna (UX): Memastikan situs dapat digunakan dan terlihat bagus di semua perangkat, mulai dari layar desktop besar hingga smartphone kecil, tanpa perlu membuat versi situs terpisah.
  • Masa Depan: Memastikan situs tetap berfungsi dengan baik meskipun ada perangkat baru dengan ukuran layar yang tidak terduga.
  • SEO (Search Engine Optimization): Google merekomendasikan RWD sebagai praktik terbaik karena lebih efisien bagi mesin pencari untuk merayapi dan mengindeks satu URL per konten.
  • Efisiensi Pengembangan: Lebih mudah dikelola dan dipelihara karena hanya ada satu basis kode untuk semua perangkat.

c. Bagaimana kombinasi CSS, JavaScript, dan prinsip RWD dapat menghasilkan tampilan web yang interaktif sekaligus adaptif? Jelaskan bagaimana ketiganya saling melengkapi!

Kombinasi ketiganya menciptakan pengalaman web modern yang lengkap:

  • CSS dan RWD (Adaptif): CSS, khususnya melalui **Media Queries**, adalah inti dari adaptasi. RWD menggunakan kueri media untuk menerapkan gaya (CSS) yang berbeda berdasarkan karakteristik perangkat (lebar layar, resolusi, dll.). Ini memastikan tata letak dan ukuran font menyesuaikan diri secara mulus di berbagai perangkat.
  • JavaScript (Interaktif): JS menambahkan fungsionalitas dan interaktivitas. Contoh: animasi saat gulir (scroll), validasi input, atau fitur yang ada di praktikum ini: menyembunyikan/menampilkan konten saat diklik.
  • Saling Melengkapi:
    • **Interaktivitas Adaptif:** JS dapat mendeteksi lebar layar dan memicu perubahan yang lebih kompleks daripada yang dapat dilakukan CSS murni (misalnya, memuat data yang berbeda atau mengubah struktur DOM).
    • **Transisi CSS dalam Interaktivitas JS:** Ketika JS mengubah properti seperti kelas (`.hidden`) untuk menyembunyikan elemen, CSS dengan properti `transition` memastikan perubahan tersebut (misalnya, menghilang) terjadi secara halus dan visual yang nyaman, bukan perubahan yang tiba-tiba.
